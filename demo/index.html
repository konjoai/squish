Squish.

Make your AI use 4× less memory without losing quality.

Squish shrinks the part of a language model that grows with the conversation — its memory cache — so you can run bigger models, hold longer conversations, and serve more users on the same hardware. One line of code switches between three compression levels.

How it works

Three steps. No model retraining.

Squish operates on the running model, not the saved file. Your weights, your training, your fine-tunes — all untouched. Squish just compresses what the model remembers as it talks.

1

Spread the load

A few "loud" numbers in the cache normally make the rest look tiny and break compression. Squish first rotates them so all values share the spotlight evenly.

2

Squeeze each number smaller

Each remembered number normally takes 16 bits. Squish rounds them to 8, 4, or even 2 bits — three settings. Bigger squish = less memory, slightly less fidelity. You pick.

3

Pack them tight

Squish then bit-packs the compressed numbers — fitting four 2-bit values into one byte, where one would normally hold a single 8-bit number. Same math, fraction of the bytes.

Snippets · copy & paste

Drop-in. One mode argument.

Squish slots into the cache slot of any mlx_lm model with one line. The hard part — picking the right mode for your context length — is automated.

Quick start — auto-pick the mode python 
from squish.kv.kv_cache import (
    HadamardKVCache,
    recommended_kv_mode_3tier,
)

# auto-pick the right mode for your planned conversation length
mode = recommended_kv_mode_3tier(32_000)   # → 'int2'

cache = HadamardKVCache(
    n_layers=28,
    window=128,
    mode=mode,
    seed=42,
)

# pass it to mlx_lm — same protocol as the default cache
model.generate(prompt, kv_cache=cache)
Measure your own quality python 
import numpy as np
from squish.kv.kv_cache import (
    _quantize_int4_per_channel,
    _dequantize_int4_per_channel,
)

v   = np.random.randn(512, 128).astype(np.float16)
q,s = _quantize_int4_per_channel(v)
vh  = _dequantize_int4_per_channel(q, s, 128)

err = np.mean((v - vh)**2)
snr = 10 * np.log10(np.mean(v**2) / err)
print(f"INT4 quality = {snr:.2f} dB")
Three-tier auto mode python 
from squish.kv.kv_cache import recommended_kv_mode_3tier

recommended_kv_mode_3tier(4_000)    # 'int8'  short chat
recommended_kv_mode_3tier(12_000)   # 'int4'  medium
recommended_kv_mode_3tier(32_000)   # 'int2'  long

# override the default thresholds if you want
recommended_kv_mode_3tier(
    12_000,
    short_threshold=4_096,
    long_threshold=32_768,
)                                  # 'int4'
Raw 2-bit codec python 
from squish.kv.kv_cache import (
    _quantize_int2_per_channel,
    _dequantize_int2_per_channel,
)

# 4-level codebook · 4 numbers fit into 1 byte
packed, scale = _quantize_int2_per_channel(activations)

# packed.shape == (n_tokens, head_dim // 4)
# packed.dtype == np.uint8                 
# scale.shape  == (n_tokens,) float32       

recon = _dequantize_int2_per_channel(packed, scale, 128)
